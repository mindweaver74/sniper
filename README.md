# 🎯 타이밍 저격

> 조준경이 너구리와 겹치는 순간 탭! 정확할수록 고득점!

**▶ 지금 바로 플레이** → [https://mindweaver74.github.io/sniper](https://mindweaver74.github.io/sniper)

---

## 🎮 게임 방법

1. 조준경이 너구리 위를 지나갑니다
2. 너구리 중앙과 겹치는 순간 **탭!**
3. 정확할수록 높은 등급과 점수
4. 3번 빗맞으면 게임 오버

---

## 🏆 등급 체계

| 정확도 | 등급 | 점수 | 너구리 반응 |
|--------|------|------|------------|
| 92% 이상 | PERFECT ✨ | 1000점 | **꿱!!** 눈 X자 + 혀 |
| 75% 이상 | GREAT | 600점 | **아야앙~** 눈물 |
| 55% 이상 | GOOD | 300점 | **아야!** 눈 꾹 |
| 30% 이상 | OK | 100점 | 무반응 |
| 그 이하 | MISS 💀 | 0점 | **메롱~** 윙크 + 혀 |

---

## 🦝 너구리 리액션

맞을 때마다 너구리가 다르게 반응해요!

- 🎯 **PERFECT** — 눈이 X자가 되며 "꿱!!" + 별☆ 이펙트
- 😢 **GREAT** — 눈물 흘리며 "아야앙~"
- 😣 **GOOD** — 눈 꾹 감고 "아야!" + 땀방울
- 😜 **MISS** — 윙크하며 혀 내밀고 "메롱~ 🤪"

---

## 📈 스테이지 진행

명중 **10발마다** 자동으로 다음 스테이지로 진행됩니다.

| 스테이지 | 조준경 속도 | 이동 범위 | 과녁 크기 |
|---------|-----------|----------|---------|
| 1 | 느림 | 좁음 | 크다 |
| 3 | 보통 | 보통 | 보통 |
| 5 | 빠름 | 넓음 | 작다 |
| 10+ | 매우 빠름 | 매우 넓음 | 매우 작다 |

스테이지가 올라갈 때마다 **"STAGE N!"** 메시지와 효과음이 나옵니다.
